# E-Commerce Website (Shopee Simulation)

**Software Engineering Capstone Project**  
A full-featured multi-seller e-commerce platform simulating Shopee, built with ASP.NET MVC.

---

## Project Overview

This project is a comprehensive e-commerce website that supports three main user roles: **Buyer**, **Seller**, and **Admin**. It covers the entire business flow from product browsing to order management, payment, and analytics.

The system was developed following a standard software development lifecycle (requirements gathering → analysis → design → implementation → testing).

**Period:** September 2025 – December 2025  
**Instructor:** Nguyen Thi Bich Ngan

---

## Key Features

### Buyer
- User registration and login
- Product search and filtering
- Product detail view
- Shopping cart management
- Order placement and Momo payment integration
- Order tracking
- Product review and rating

### Seller
- Shop management
- Add, edit, and delete products
- Order management (confirm, ship, cancel)
- Revenue and sales statistics

### Admin
- User and shop management
- Product approval and moderation
- System-wide order management
- Sales reports and analytics
- Promotion and review management

---

## Technologies Used

- **Backend:** ASP.NET MVC 5
- **Database:** Microsoft SQL Server
- **Frontend:** HTML5, CSS3, Bootstrap, JavaScript, jQuery
- **ORM:** Entity Framework
- **Authentication:** ASP.NET Identity
- **Payment Gateway:** Momo
- **Tools:** Visual Studio, SQL Server Management Studio, Figma, Draw.io
